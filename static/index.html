Snuggle is designed to help you identify desirable new Wikipedia editors. The interface lists all accounts registered in the last 30 days along with contributions and talk page messages.

New users

  1. User account metadata and statistics.
  2. Interactive graph of user contribs for the first 30 days. Different colors represent different namespaces. E.g., Articles, Talk, User, User_talk, Wikipedia, Wikipedia_talk, etc. Revisions reverted by another editor are noted with a dot ().
  3. Talk page activity. Templated messages are noted with an icon. E.g.
  4. Categorization buttons. This will move the user into one of the category lists.

Category lists

Clicking on the tabs on the top of the interface allows you to view different categories of newcomers.

Controls

A set of filters allow you to limit and sort the users that are listed in the interface.

Keyboard shortcuts

While you can use the mouse to peform some actions, you can use a set of keyboard shortcuts to peform some common actions at high speed.

User selection
Page Up moves up and Page Down moves down
Revision selection (in the graph)
The arrow keys: ↑, →, ↓ and ←
Categorize the selected user
Number keys: #1 for "good-faith", #2 for "ambiguous" and #3 for "bad-faith"
