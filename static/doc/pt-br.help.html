Snuggle versão: 1.1

Guia Rápido

Veja a documentação completa aqui

O Snuggle foi desenvolvido para ajudá-lo a identificar editores novatos desejáveis. A interface lista todas os novos usuários registrados nos últimos 30 dias junto de suas contribuições e mensagens em páginas de discussão.

Interface do usuário

Figura 1. Controles.
  1. Lista de usuários selecionados manualmente
    • não categorizado - Novos usuários que ainda não foram categorizados.
    • boa fé - Novos usuários que está pelo menos tentando fazer algo produtivo.
    • ambíguo - Novos usuários cujas motivações não estão claras.
    • má fé - Novos usuários que aparentam não estarem tentando ser produtivos.
  2. Número mínimo de edições.
  3. O domínio onde o mínimo de edições foram feitas.
  4. Critérios para ordenar usuários
    • desejabilidade - Índice gerado automaticamente com a probabilidade do usuário estar editando com boa fé.
    • data de registro - Data de registro do usuário.
    • última atividade - Data da última edição do usuário.
    • edições revertidas - Número de edições do usuário que foram revertidas.
    • total de edições - Número total de edições do usuário.
  5. Direção para ordenar usuários. Isso lhe permite inverter a order de qualquer critério de ordenação.
Figura 2. Usuário Expandido.
  1. Metados e estatísticas da conta de usuário.
    • Nome de usuário
    • Nome - Discussão - Contribuições
    • Data de registro
    • Última atividade
    • # de vezes visto no Snuggle
    • Gráfico com número de revisões
    • # de edições revertidas (feitas por outro usuário)
  2. Atividade da Página de Discussão. Certos tipos de mensagem são marcadas com um ícone e.g.:
    • Eliminação semirrápida
    • Avisos de nível 2
    • Mensagem de boas vindas
    • Eliminação rápida
    • Discussões gerais
  3. Botões de categorização. Eles irão mover o usuário para a lista de uma categoria (explicadas abaixo).
  4. Gráfico interativo de contribuições nos primeiros 30 dias. Cores diferentes representam diferentes domínios. E.g.:
    • Principal
    • Discussão
    • Usuário(a)
    • Usuário(a) Discussão
    • Wikipédia
    • Wikipédia Discussão
    Revisões revertidas por outro editor são marcadas com um ponto (). Auto reversões tem um ponto vazado ()

Atalhos de teclado

Você pode utilizar o mouse para realizar todas as ações, mas também dispõe de atalhos de teclado para realizar algumas ações comuns em velocidade maior.

Seleção de usuário
Page Up move para cima e Page Down move para baixo
Seleção de revisão (no gráfico)
As setas: ↑, →, ↓ e ←
Categorizar o usuário selecionado
Teclas numéricas: #1 para "boa fé", #2 para "ambíguo" e #3 para "má fé"
